# streamlits

streamlit cloudのソースコードのページです。
streamlitのマルチページを使って複数のstreamlit appを動かします。

## Runs on

- StreamlitCloud

    see: https://syakesaba.streamlit.app/

- Your PC(localhost)

    ```sh
    git clone https://github.com/syakesaba/streamlits
    cd streamlits
    uv init
    uv sync
    uv run stremalit run 🏠️.py
    ```
